# Razorpay Payment Gateway Demo

A simple Node.js + Express demo project showcasing Razorpay Checkout integration in test mode.  
It covers order creation, client-side checkout, and server-side payment verification.

## Features
- Create Razorpay orders
- Checkout integration (Test Mode)
- Secure payment verification using signature
- Success & failure handling

## Tech Stack
- Node.js
- Express
- Razorpay SDK
- EJS / HTML
- JavaScript

## Setup
1. Clone the repo
2. Run `npm install`
3. Create a `.env` file and add:
   - RAZORPAY_KEY_ID
   - RAZORPAY_KEY_SECRET
4. Run `npm start`
5. Open `http://localhost:3000`
